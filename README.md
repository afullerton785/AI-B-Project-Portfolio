# Reimagining Football Heritage through Human-Artificial Intelligence Collaboration 

For this project I selected a historical football pamphlet from the University of Glasgow Special Collections. The pamphlet contained illustrations which I modernised using AI, and got some of my peers to recreate them in real life. The aim for this project is to explore how humans and Artificial Intelligence (AI) can reinterpret items of cultural heritage.

This repository is split into four seperate Jupyter notebooks: Context, AI_Generation, Human_Recreation, and Reflection. These each explore an integral part of this project. Context will explore the pamphlet itself and the original images, AI_Generation will explore the steps taken with AI to modernise the illustrations, Human_Recreation will highlight the ways humans recreated the images, and Reflection will contain an analysis of the two side-by-side.
